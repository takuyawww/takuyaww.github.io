## コードブロックのテスト

### JavaScript

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome, ${name}`;
}

const result = greet("World");
console.log(result);
```

### Python

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

for i in range(10):
    print(f"fib({i}) = {fibonacci(i)}")
```

### TypeScript

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

const users: User[] = [
  { id: 1, name: "Alice", email: "alice@example.com" },
  { id: 2, name: "Bob", email: "bob@example.com" }
];

function findUser(id: number): User | undefined {
  return users.find(user => user.id === id);
}
```

## リストのテスト

### 順序なしリスト

- 項目1
- 項目2
- 項目3
  - ネストされた項目1
  - ネストされた項目2

### 順序付きリスト

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
